# TempoTap Support

Static support pages for TempoTap.

## Pages

- `index.html` - support hub
- `privacy.html` - Privacy Policy URL for App Store Connect
- `support.html` - Support URL for App Store Connect
- `licenses.html` - asset credits and license notices

## GitHub Pages

Enable GitHub Pages for this public repository, then use:

- Privacy Policy URL: `https://<your-github-name>.github.io/tempotap-support/privacy.html`
- Support URL: `https://<your-github-name>.github.io/tempotap-support/support.html`

If the repository name is different, replace `tempotap-support` with the actual repository name.
